## 📖 About the Project

This repository contains the design source for a digital agency landing page, created to demonstrate how a creative agency could present its work, build trust with potential clients, and drive project inquiries through a clean and modern web presence.

The page is built around a simple but effective narrative: introduce the agency, prove credibility through trusted brand logos and past project work, reinforce trust further with client testimonials, and finally guide the visitor toward starting a project. Every section was designed with a clear purpose, avoiding unnecessary clutter so visitors can quickly understand what the agency offers and why they should choose to work with it.

The overall tone of the design leans soft and approachable in the hero and project sections, while the closing "Let's Work Together" section shifts to a bold, high-contrast dark theme — creating a strong visual close to the page and drawing attention to the final call to action.

## ✨ Features

The landing page is structured into focused, purposeful sections that guide a visitor from first impression to action:

- **Hero Section** — A bold introductory headline ("We create the best digital projects") paired with a supporting tagline and a prominent **"Start Your Project"** call-to-action button, along with a hero image showing a creative team at work.
- **Trusted By Section** — A row of recognizable brand logos (Google, Instagram, Pinterest, YouTube) to immediately establish credibility and trust with visitors.
- **Discover Our Best Projects Section** — A grid-based showcase of recent project work, each entry paired with a project image, short description, and a **"View Projects"** link to encourage deeper exploration of the agency's portfolio.
- **Testimonial Section** — Client reviews displayed with star ratings, reviewer photos, names, and roles, reinforcing trust through real (or representative) client feedback.
- **Call-to-Action Section ("Let's Work Together")** — A bold, dark-themed closing section designed to capture attention and prompt visitors to start a project, paired with a repeated **"Start Your Project"** button.
- **Footer** — Clean navigation links (Projects, Courses, About Us, Contact Us), social media icons (YouTube, Instagram, Twitter), and copyright information.
- **Consistent Navigation Bar** — A sticky top navigation with the agency name/logo, key page links, and a persistent "Start Your Project" button for quick access from anywhere on the page.
- **Fully Responsive Layout** — The design adapts cleanly across desktop, tablet, and mobile screen sizes, maintaining readability and visual balance at every breakpoint.

## 🛠️ Designed With

- **Figma** — UI/UX design and prototyping

### Figma Features Used

This project was designed entirely in Figma, with the following features used to build a polished, consistent, and developer-ready design:

- **Auto Layout** — Used to maintain consistent spacing, padding, and alignment across sections like the project grid and testimonial cards, so the layout stays balanced even as content changes.
- **Components & Variants** — Repeated UI elements such as buttons, project cards, testimonial cards, and navigation links were built as reusable components with variants, keeping the design consistent throughout the page.
- **Prototyping / Interactive Flows** — A clickable prototype was created to simulate the navigation flow between the hero CTA, project links, and the final "Start Your Project" section, helping validate the user journey before development.
- **Responsive / Constraints** — Layout constraints were applied to ensure elements like the hero image, project grid, and testimonial cards resize and reposition predictably across different screen sizes.
- **Design System** — A consistent design system — including the soft pink/blue color palette, typography styles, button states, and spacing rules — was maintained across every section to keep the page cohesive and professional.

## 🎯 Project Goals

The core goals behind this landing page design were to:

- Present a digital agency's value proposition clearly within the first few seconds of landing on the page
- Build trust quickly through recognizable brand logos and authentic-feeling client testimonials
- Showcase past project work in a clean, scannable grid format
- Guide visitors naturally toward a single, clear action: starting a project
- Create a responsive, professional design that reflects strong UI/UX fundamentals
